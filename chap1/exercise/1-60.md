# 演習問題1.60

引き算は、負の数を加算するとして扱う。

## (a)
9 - 7 = 9 + (-7) と変換する。

5ビットの2の補数の和の計算とすると
```
  0_1001
+)1_1001
  ------
 10_0010
```
正と負の和の時は、桁溢れは無視してよいので、桁溢れは発生しない

## (b)
```
  0_1100
+)1_0001
  ------
  1_1101
```
桁溢れは発生しない

## (c)
```
  1_1010
+)1_0101
  ------
 10_1111
```
桁溢れが発生する

## (d)
```
  0_0100
+)0_1000
  ------
  0_1100
```
桁溢れは発生しない
