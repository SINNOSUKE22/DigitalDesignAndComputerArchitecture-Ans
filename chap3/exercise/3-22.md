記載# 演習問題3.22

バイナリ・エンコーディングを用いない状態遷移表は以下の通り。

```
 S   A  B |  S'
----------+----
 S0 ~A  X | S0 
 S0  A  X | S1 
 S1  X ~B | S0 
 S1  X  B | S2 
 S2  X  X | S0 
```

これは、Aの後に、~BではなくてBが出た時に1を出力するマシン。

S0〜S2の状態を以下のようにエンコーディングする。
```
    S_1 S_2
S0:  0   0
S1:  0   1
S2:  1   0
```

バイナリ・エンコーディングを用いた状態遷移表は以下の通り。
```
 S_1 S_0 A B | S_1' S_0'
-------------+----------
  0   0  1 X |  0    1
  0   0  0 X |  0    0
  0   1  X 1 |  1    0
  0   1  X 0 |  0    0
  1   0  X X |  0    0
```

出力表は、以下の通り。
```
 S_1 S_0 | Q
---------+---
  0   0  | 0
  0   1  | 0
  1   0  | 1
```

状態遷移のカルノーマップは以下の通り。
```			
S_0'                     
    |       S_1:0       |
    | 00 | 01 | 11 | 10 |
----+----+----+----+----+
 00 |  0 |  0 |  X |  0 |
 01 |  0 |  0 |  X |  0 |
    |+--+|    |	   |   	|
 11 || 1||  0 |  X |  0 |
 10 || 1||  0 |  X |  0 |
     +--+

S_1'                     
    |       S_1:0       |
    | 00 | 01 | 11 | 10 |
----+----+----+----+----+    
 00 |  0 |  0 |  X |  0 |
    |  	 |+-------+|   	|
 01 |  0 || 1 |  X||  0 |
 11 |  0 || 1 |  X||  0 |
    |    |+-------+|    |
 10 |  0 |  0 |  X |  0 |
```
