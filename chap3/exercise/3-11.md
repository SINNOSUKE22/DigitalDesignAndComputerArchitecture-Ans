# 演習問題3.11

真理値表は以下の通り。

```
 A B |   C
-----+-------
 0 0 |   0
 0 1 | Cpre
 1 0 | Cpre
 1 1 |   1
```

AとBが同じならその値、
違っていたら、前の値のまま。

FPGAで同期式の設計をすると、消費電力が増加する問題が発生する。

この問題を解決する方法として、非同期式設計が提案されている。この設計方式の重要な部分を担うのがCエレメント
